# Latex voor gevorderden

Latex voor gevorderden is een vervolg op `latex-voor-beginners` met uitwerkingen van voorbeelden op de volgende gebieden;     

* [Tabellen](tabellen.md)  
* Wiskunde
* Grafieken
* Tekenen
* Programmeer code

In tegenstelling tot `Latex voor beginners` gaat `Latex voor gevorderden` dieper in op de onderwerpen, verder uitgewerkt in de besbetreffende wiki pagina's.
De diverse onderwerpen worden zoveel mogelijk onafhankelijk van elkaar gepresenteerd en in `.tex` bestanden uitgwerkt.
